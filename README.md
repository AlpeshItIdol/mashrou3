# mashrou3

<h1>Mashrou3</h1>

Mashrou3 is a real estate platform that facilitates the buying, selling, and financing of properties. 
It caters to different user types—visitors, vendors, real estate owners, bank admins, super admin 
and admin staff by providing a comprehensive suite of tools for property management, service offers,
and financial services. 

## **Features**<br>

**Authentication:** Secure login process with mobile number authentication. Users select a country 
(default: Jordan), enter their mobile number, receive a verification code (OTP), and log in.

**Role-Based Dashboard:** The app supports three user roles – Owner, Vendor, and Visitor. Each role 
has access to specific functionalities. Visitors have a bottom navigation bar with Home, Favorites, 
and Notifications.

**Home Screen (Visitor):** Users can browse a list of properties, perform searches, apply filters, 
sort results, and view properties categorized into Commercial, Residential, and Land etc.

**Property Listings:** Property widgets display images, name, location, price, and availability, 
with a favorite button to add properties to the favorites list.

**Property Details:** Users can view property images and GIFs, price, title, description, size, 
location, and review count. Additional features include video walkthroughs, 3D virtual tours, Google
Maps integration, nearby facilities, and amenities. Users can also contact the owner via call and 
share property details.

**Profile Management:** Users can view and update their profile details, including personal 
information and preferences.

**Owner Features:** Property owners can add, edit, and delete properties, manage visit requests, 
and handle inquiries from visitors.

**Vendor Features:** Vendors can create, edit, and delete offers to promote properties.

**Banking Information:** Users can view a list of partnered banks offering real estate financing 
options and mortgage plans.

**Visit Requests:** Visitors can request property visits, and owners can approve or reject requests.

**Notifications & Alerts:** Users receive real-time notifications for visit requests, offers, 
property updates, and promotions.

**Drawer Menu:** Includes options for Personal Information, Recently Visited Properties, Requested 
Properties, Properties with Offers, Terms & Conditions, Support, and App Preferences (language 
change and light/dark mode toggle).

Overall, the app provides a comprehensive and user-friendly real estate solution, ensuring an 
efficient experience for Owners, Vendors, and Visitors.

## **Prerequisites**<br>

Before running the app, make sure you have the following:

Android Studio installed on your machine.
Get the flutter SDK.
An internet connection to handle authentication.
Google map API key.
